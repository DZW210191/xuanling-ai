# 模式和最佳实践

## 记忆系统架构

### 双层记忆结构

1. **长期记忆 (MEMORY.md)**
   - 重要性 ≥4 分的内容
   - 永久保存
   - 核心知识库

2. **短期记忆 (memory/YYYY-MM-DD.md)**
   - 重要性 2-3 分的内容
   - 30 天周期，时间衰减

### 专题文件

| 文件 | 用途 |
|------|------|
| contacts.md | 联系人信息 |
| decisions.md | 重要决策 |
| preferences.md | 用户偏好 |
| projects.md | 项目信息 |
| patterns.md | 模式和最佳实践 |
| feedback.md | 反馈记录 |

## 检索机制

- 每次回答前自动搜索相关记忆
- 关键词 + 语义搜索
- 时间衰减：30天周期

## 写入策略

| 重要性 | 写入位置 |
|--------|----------|
| ≥4 分 | MEMORY.md |
| 2-3 分 | 当日日志 |
| <2 分 | 不记录 |

## 触发词

- "记下来"、"记住这个" → 永久保存
- "别忘了" → 提醒
- "这是一个重点" → 高重要性
