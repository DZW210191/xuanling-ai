- 审计日志记录和查询

## 新增 API 路由

### Skills
- `GET /api/skills` - 列出技能
- `GET /api/skills/{name}` - 技能详情
- `POST /api/skills/{name}/execute` - 执行技能
- `POST /api/skills/reload` - 热重载

### SubAgents
- `GET /api/subagents` - 列出代理
- `POST /api/subagents` - 创建代理
- `GET /api/tasks` - 任务列表
- `POST /api/tasks` - 提交任务
- `POST /api/tasks/plan` - 规划任务

### Memory
- `POST /api/memory` - 创建记忆
- `GET /api/memory/search` - 语义搜索
- `GET /api/memory/working` - 工作记忆

### Security
- `GET /api/security/users` - 用户列表
- `POST /api/security/api-keys` - 创建密钥
- `GET /api/security/audit-logs` - 审计日志
