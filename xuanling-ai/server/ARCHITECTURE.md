## 核心组件

### 1. AI 对话引擎
- 支持多种 LLM (MiniMax, OpenAI, Claude 等)
- Function Calling 支持
- 上下文管理和记忆注入

### 2. 工具系统
每个工具定义:
```python
{
    "name": "read",
    "description": "读取文件内容",
    "parameters": {
        "path": {"type": "string", "description": "文件路径"}
    },
    "handler": callable
}
```

### 3. Skills 系统
- 动态加载 Python 模块
- 自动注册工具
- 支持热重载

### 4. 子代理系统
- 任务队列管理
- 并行执行
- 状态监控

### 5. 记忆系统
- 向量化存储 (使用 sentence-transformers)
- 语义搜索
- 重要性评分

### 6. 安全沙箱
- 工具权限控制
- 命令执行白名单
- 资源使用限制
