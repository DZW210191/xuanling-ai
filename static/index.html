💻 本地运行指南

快速开始

# 1. 克隆项目
git clone https://github.com/DZW210191/xuanling-ai.git
cd xuanling-ai

# 2. 安装依赖
pip install -r requirements.txt

# 3. 配置环境变量
export API_KEY="你的API_KEY"
export API_BASE_URL="https://code.coolyeah.net/v1"

# 4. 运行
python src/main.py

或使用 Docker

# 构建并运行
docker build -t xuanling-ai .
docker run -p 8000:8000 -e API_KEY=你的KEY xuanling-ai

访问地址

启动后访问: http://localhost:8000
